## Overview

The Text-to-Speech API converts text into high-quality audio using ElevenLabs' multilingual voice synthesis. It supports multiple voices and returns audio as playable blob URLs.

## Function Reference

### `textToSpeech()`

Converts text to speech and returns an audio URL.

<ParamField path="text" type="string" required>
  The text to convert to speech
</ParamField>

<ParamField path="voiceId" type="string" default="eYO9Ven76ACQ8Me4zQK4">
  ElevenLabs voice ID. Defaults to a preset voice if not specified.
</ParamField>

<ResponseField name="audio_url" type="string">
  Blob URL pointing to the generated audio file (MP3 format)
</ResponseField>

<ResponseField name="null" type="null">
  Returns `null` if an error occurs during processing
</ResponseField>

## Configuration

### API Settings

<ParamField path="NEXT_PUBLIC_ELEVEN_API_KEY" type="string" required>
  ElevenLabs API key stored in environment variables
</ParamField>

<ParamField path="model_id" type="string" default="eleven_multilingual_v2">
  ElevenLabs voice model
</ParamField>

<ParamField path="output_format" type="string" default="mp3_22050_32">
  Audio format and quality (22.05kHz, 32kbps MP3)
</ParamField>

### Voice Settings

<ParamField path="similarity_boost" type="number" default="0.9">
  Voice similarity enhancement (0-1). Higher values make the voice more consistent.
</ParamField>

<ParamField path="stability" type="number" default="1">
  Voice stability (0-1). Higher values produce more consistent output.
</ParamField>

```typescript
import textToSpeech from './api/textToSpeech';

// Convert text to speech with default voice
```

## Code Reference

Source: `src/pages/api/textToSpeech.ts:1`

```typescript
async function textToSpeech(text: string, voiceId: string = 'eYO9Ven76ACQ8Me4zQK4') {
  const options = {
    method: 'POST',
    headers: {
      'xi-api-key': process.env.NEXT_PUBLIC_ELEVEN_API_KEY || '',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model_id: "eleven_multilingual_v2",
      text: text,
      voice_settings: {
        similarity_boost: 0.9,
        stability: 1
      }
    })
  };

  try {
    const response = await fetch(`https://api.elevenlabs.io/v1/text-to-speech/${voiceId}?output_format=mp3_22050_32`, options);

    if (response.ok && response.headers.get("content-type")?.includes("audio")) {
      const audioBlob = await response.blob();
      const audioUrl = URL.createObjectURL(audioBlob);
      return { audio_url: audioUrl };
    } else {
      throw new Error('Network response was not ok or the content type is not audio');
    }
  } catch (err) {
    console.error(err);
    return null;
  }
}
```

## Response Handling

### Success Response

```typescript
{
  audio_url: "blob:http://localhost:3000/a1b2c3d4-e5f6-7890"
}
```

The blob URL can be used with:
- HTML `<audio>` elements
- JavaScript `Audio()` constructor
- Media player libraries

### Error Response

Returns `null` when:
- API key is invalid
- Network request fails
- Response content type is not audio
- Voice ID doesn't exist

## Available Voices

The default voice ID is `eYO9Ven76ACQ8Me4zQK4`. To use different voices:

1. Visit [ElevenLabs Voice Library](https://elevenlabs.io/voice-library)
2. Select a voice and copy its ID
3. Pass the ID as the second parameter

## Environment Setup

Add your ElevenLabs API key to `.env.local`:

```bash
NEXT_PUBLIC_ELEVEN_API_KEY=your_api_key_here
```

## Best Practices

### Memory Management

Revoke blob URLs when done to free memory:

```typescript
const result = await textToSpeech("Hello");
if (result) {
  const audio = new Audio(result.audio_url);
  audio.play();
  
  audio.onended = () => {
    URL.revokeObjectURL(result.audio_url);
  };
}
```

### Error Handling

Always check for null responses:

```typescript
const result = await textToSpeech(text);
if (!result) {
  console.error('Failed to generate speech');
  // Show user-friendly error message
  return;
}
```

### Rate Limiting

ElevenLabs has rate limits based on your plan. Implement queuing for multiple requests:

```typescript
const queue = [];
let processing = false;

async function queueSpeech(text: string) {
  queue.push(text);
  if (!processing) {
    await processQueue();
  }
}
```

## Audio Format

- **Format**: MP3
- **Sample Rate**: 22.05 kHz
- **Bitrate**: 32 kbps
- **Quality**: Optimized for voice clarity and file size

To change format, modify the `output_format` query parameter in the API URL.
