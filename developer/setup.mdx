## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js 20+ and npm/yarn
- Python 3.8+ and pip
- Git

## Installation

<Steps>
  <Step title="Clone the Repository">
    Clone the EliteCode repository to your local machine:
    
    ```bash
    git clone <repository-url>
    cd elitecode
    ```
  </Step>

  <Step title="Install Frontend Dependencies">
    Install all required Node.js packages:
    
    ```bash
    npm install
    # or
    yarn install
    ```

    ### Key Dependencies

    The project uses the following major dependencies:

    <CodeGroup>
    ```json package.json (excerpt)
    {
      "dependencies": {
        "next": "14.1.3",
        "react": "^18.2.0",
        "react-dom": "^18.2.0",
        "openai": "^4.33.0",
        "@radix-ui/react-dialog": "^1.0.5",
        "framer-motion": "^11.0.25",
        "tailwindcss": "^3.3.0",
        "typescript": "^5"
      }
    }
    ```
    </CodeGroup>
  </Step>

  <Step title="Install Python Dependencies">
    Install the Python backend dependencies:
    
    ```bash
    pip install -r requirements.txt
    ```

    The `requirements.txt` includes:
    - `flask` - Web framework for the backend API
    - `whisper` (from OpenAI GitHub) - Speech-to-text transcription
    - `setuptools-rust` - Build tools for Whisper
    - `flask_cors` - Cross-origin resource sharing support
  </Step>

  <Step title="Configure Environment Variables">
    Create a `.env.local` file in the root directory with the following variables:

    ```bash .env.local
    # OpenAI API Key for GPT completions and text-to-speech
    NEXT_PUBLIC_OPENAI_API_KEY=your_openai_api_key_here

    # Backend API URL (default: http://localhost:5000)
    NEXT_PUBLIC_API_URL=http://localhost:5000
    ```

    <Warning>
      The `NEXT_PUBLIC_` prefix exposes these variables to the browser. Never commit actual API keys to version control.
    </Warning>
  </Step>
</Steps>

## Running the Development Server

### Start the Backend (Flask)

In one terminal, start the Python Flask server:

```bash
cd src/python
python app.py
```

The Whisper API will be available at `http://localhost:5000`.

### Start the Frontend (Next.js)

In another terminal, start the Next.js development server:

```bash
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:3000`.

| `npm start` | Start the production server |
| `npm run lint` | Run ESLint to check code quality |

## Verify Installation

Once both servers are running:

1. Open `http://localhost:3000` in your browser
2. Grant microphone permissions when prompted
3. Click on any data structure card (e.g., "array", "linked list")
4. Click the microphone icon to test voice recording
5. Speak an explanation of the topic
6. The AI should respond with feedback

<Check>
  If you can record audio and receive an AI response, your development environment is set up correctly!
</Check>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Microphone not working">
    - Ensure you've granted browser microphone permissions
    - Check that `MediaRecorder` API is supported in your browser
    - Try using Chrome/Edge for best compatibility
  </Accordion>

  <Accordion title="Whisper API connection failed">
    - Verify the Flask server is running on port 5000
    - Check that `NEXT_PUBLIC_API_URL` is set correctly
    - Ensure CORS is enabled in the Flask backend
  </Accordion>

  <Accordion title="OpenAI API errors">
    - Verify your `NEXT_PUBLIC_OPENAI_API_KEY` is valid
    - Check your OpenAI account has available credits
    - Ensure the API key has permissions for chat completions
  </Accordion>
</AccordionGroup>
