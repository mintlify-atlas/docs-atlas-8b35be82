## Overview

EliteCode's learning modules provide an interactive, visually engaging interface for exploring data structures and algorithms topics. The system uses a Bento Grid layout for topic selection and modal popups for immersive learning experiences.

## Topic Selection Interface

### Bento Grid Layout

The homepage features a dynamic grid of topic cards powered by the `BentoGrid` component:

```tsx BentoGrid.tsx
export const BentoGrid = ({
  className,
  children,
}: {
  className?: string;
  children?: React.ReactNode;
}) => {
  return (
    <div
      className={cn(
        "grid md:auto-rows-[18rem] grid-cols-1 md:grid-cols-3 gap-4 max-w-7xl mx-auto ",
        className
      )}
    >
      {children}
    </div>
  );
};
```

<Info>
The grid automatically adapts to screen size: single column on mobile, three columns on desktop with 18rem row height.
</Info>

## Topic Card Component

### BentoGridItem Structure

Each topic card is rendered using the `BentoGridItem` component:

```tsx BentoGrid.tsx
export const BentoGridItem = ({
  className,
  title,
  description,
  header,
  icon,
}: {
  className?: string;
  title?: string | React.ReactNode;
  description?: string | React.ReactNode;
  header?: React.ReactNode;
  icon?: React.ReactNode;
}) => {
  return (
    <div
      className={cn(
        "row-span-1 rounded-xl group/bento hover:shadow-xl transition duration-200 shadow-input dark:shadow-none p-2 dark:bg-black dark:border-white/[0.2] bg-white border border-transparent justify-between flex flex-col space-y-3",
        className
      )}
    >
      {header}
      <div className="group-hover/bento:translate-x-2 transition duration-200">
        {icon}
        <div className="font-sans font-bold text-neutral-600 dark:text-neutral-200 mb-2 mt-2">
          {title}
        </div>
        <div className="font-sans font-normal text-neutral-600 text-xs dark:text-neutral-300">
          {description}
        </div>
      </div>
    </div>
  );
};
```

### Hover Animations

<Tip>
Cards feature smooth hover effects:
- Shadow expands on hover (`hover:shadow-xl`)
- Content slides right 2px (`group-hover/bento:translate-x-2`)
- 200ms transition duration for smooth animation
</Tip>

## Dynamic Grid Sizing

The `Homebento` component implements variable sizing for visual interest:

```tsx BentoGrid.tsx
export function Homebento() {
  return (
    <BentoGrid className="w-full h-full">
      {items.map((item, i) => {
        const colSpan =
          i % 2 === 0
            ? "lg:col-span-2"
            : i % 5 === 0
            ? "lg:col-span-2"
            : "lg:col-span-1";
        const rowSpan = i % 3 === 0 ? "sm:row-span-2" : "sm:row-span-1";
        
        return (
          <BentoGridItem
            key={i}
            title={item.title}
            description={item.description}
            header={item.header}
            icon={item.icon}
            className={`${
              i === 0 ? "lg:col-span-3 " : `opacity-50 ${colSpan} ${rowSpan}`
            }`}
          />
        );
      });
    </BentoGrid>
  );
}
```

### Sizing Rules

| Index Pattern | Column Span | Row Span | Effect |
|---------------|-------------|----------|--------|
| `i === 0` | 3 columns | Default | Featured first card spans full width |
| `i % 2 === 0` | 2 columns | Default | Even cards are wider |
| `i % 3 === 0` | Default | 2 rows | Every third card is taller |
| `i % 5 === 0` | 2 columns | Default | Every fifth card is wider |
| Other | 1 column | 1 row | Standard card size |

<Note>
Non-featured cards have 50% opacity to emphasize the primary topic (Array).
</Note>

## Module Popup Component

### Modal Structure

When a user selects a topic, the `ChatCard` modal appears:

```tsx ModulePopup.tsx
export const ChatCard: React.FC<{ onClose: () => void; topic: string }> = ({
  onClose,
  topic,
}) => {
  const [isRecording, setIsRecording] = useState(false);
  const [isLoading, setIsLoading] = useState<boolean>(false);
  const audioRef = useRef<HTMLAudioElement>(null);

  return (
    <div
      className="fixed inset-0 z-50 bg-black bg-opacity-50 flex justify-center items-center"
      onClick={onClose}
    >
      <div
        className="relative w-2/4 h-3/4 bg-white dark:bg-black rounded-lg shadow-lg p-4 flex flex-col"
        onClick={handleCardClick}
      >
        <button
          onClick={onClose}
          className="absolute top-0 right-0 m-4 text-4xl"
        >
          &times;
        </button>
        {!isLoading && (
          <p className="flex justify-center items-center pt-28 text-xl text-medium">
            Explain to me what {topic} is
          </p>
        )}
        <Chatbox messages={[]} />
        <Recorder
          uploadAudio={uploadAudio}
          isRecording={isRecording}
          toggleRecording={toggleRecording}
        />
        <audio ref={audioRef} controls style={{ visibility: "hidden" }}></audio>
      </div>
    </div>
  );
};
```

### Modal Features

<Steps>
  <Step title="Overlay Background">
    Semi-transparent black overlay (`bg-black bg-opacity-50`) covers the entire viewport at z-index 50.
  </Step>
  
  <Step title="Centered Modal">
    Modal is 50% viewport width and 75% height, centered using flexbox.
  </Step>
  
  <Step title="Click Outside to Close">
    Clicking the overlay closes the modal, while clicks inside the modal are captured by `handleCardClick`.
  </Step>
  
  <Step title="Close Button">
    Large × button in top-right corner provides clear exit option.
  </Step>
</Steps>

## Audio Playback

### Hidden Audio Player

The modal includes a hidden HTML5 audio element for AI responses:

```tsx ModulePopup.tsx
const uploadAudio = async (blob: Blob) => {
  // ... transcription and OpenAI processing ...
  
  const ttsResponse = await textToSpeech(openAIResponse);
  if (ttsResponse) {
    const audioPlayer = audioRef.current;
    if (audioPlayer) {
      audioPlayer.src = ttsResponse.audio_url;
      audioPlayer
        .play()
        .catch((error) => console.error("Error playing audio:", error));
    }
  }
};

// Hidden audio element
<audio ref={audioRef} controls style={{ visibility: "hidden" }}></audio>
```

<Tip>
The audio element is hidden with CSS (`visibility: hidden`) rather than display:none to allow programmatic playback control.
</Tip>
