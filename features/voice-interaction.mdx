## Overview

EliteCode's voice interaction system enables users to explain data structures and algorithms concepts through natural speech. The system combines browser-based audio recording, Python-powered Whisper AI transcription, and real-time audio visualization.

## Architecture

The voice interaction flow involves three key components:

1. **Frontend Recording** - Browser MediaRecorder API captures audio
2. **Backend Transcription** - Flask server with Whisper AI processes audio
3. **Visual Feedback** - Canvas-based frequency visualization

## Audio Recording

### MediaRecorder Implementation

The `Recorder` component handles microphone access and audio capture:

```tsx Recorder.tsx
const getMicrophonePermission = async () => {
  if ("MediaRecorder" in window) {
    try {
      const streamData = await navigator.mediaDevices.getUserMedia({
        audio: true,
        video: false,
      });
      setPermission(true);
      setStream(streamData);
    } catch (err: any) {
      alert(err.message);
    }
  } else {
    alert("The MediaRecorder API is not supported in your browser.");
  }
};
```

<Note>
The system uses `audio/webm` MIME type for cross-browser compatibility.
</Note>

### Recording State Management

The recording process manages three key states:

<Steps>
  <Step title="Start Recording">
    Creates a new `MediaRecorder` instance with the audio stream and begins capturing audio chunks.
    
    ```tsx Recorder.tsx
    const startRecording = async () => {
      const media = new MediaRecorder(stream, { mimeType: mimeType });
      mediaRecorder.current = media;
      mediaRecorder.current.start();
      
      let localAudioChunks: Blob[] = [];
      mediaRecorder.current.ondataavailable = (event) => {
        if (typeof event.data === "undefined") return;
        if (event.data.size === 0) return;
        localAudioChunks.push(event.data);
      };
      setAudioChunks(localAudioChunks);
    };
    ```
  </Step>
  
  <Step title="Stop Recording">
    Stops the recorder and creates a Blob from accumulated audio chunks.
    
    ```tsx Recorder.tsx
    const stopRecording = () => {
      mediaRecorder.current.stop();
      mediaRecorder.current.onstop = () => {
        const audioBlob = new Blob(audioChunks, { type: mimeType });
        const audioUrl = URL.createObjectURL(audioBlob);
        setAudio(audioUrl);
        uploadAudio(audioBlob);
        setAudioChunks([]);
      };
    };
    ```
  </Step>
  
  <Step title="Upload Audio">
    Sends the audio blob to the Flask backend for Whisper transcription.
    
    ```tsx ModulePopup.tsx
    const uploadAudio = async (blob: Blob) => {
      const formData = new FormData();
      formData.append("audio", blob, "audio.webm");
      
      const response = await fetch(
        `${process.env.NEXT_PUBLIC_API_URL}/whisper`,
        {
          method: "POST",
          body: formData,
        }
      );
      
      const result = await response.json();
      const text = result?.results?.[0]?.transcript ?? 
        "Could not hear user, prompt an error message.";
    };
    ```
  </Step>
</Steps>

## Whisper AI Transcription

### Backend Implementation

The Flask server processes audio using OpenAI's Whisper model:

```python app.py
from flask import Flask, abort, request
from flask_cors import CORS
from tempfile import NamedTemporaryFile
import whisper
import torch

# Check if NVIDIA GPU is available
DEVICE = "cuda" if torch.cuda.is_available() else "cpu"

# Load the Whisper model
model = whisper.load_model("base", device=DEVICE)

@app.route('/whisper', methods=['POST'])
def whisper_handler():
    if not request.files:
        abort(400)
    
    results = []
    for filename, handle in request.files.items():
        temp = NamedTemporaryFile()
        handle.save(temp)
        result = model.transcribe(temp.name)
        results.append({
            'filename': filename,
            'transcript': result['text'],
        })
    
    return {'results': results}
```

<Info>
**GPU Acceleration**: The backend automatically uses CUDA if an NVIDIA GPU is available, falling back to CPU processing otherwise.
</Info>

### Model Configuration

| Parameter | Value | Description |
|-----------|-------|-------------|
| Model Size | `base` | Balances accuracy and speed |
| Device | `cuda` or `cpu` | Automatic GPU detection |
| Input Format | Audio file | Supports various formats |
| Output | JSON | Contains transcript text |

## Audio Visualization

### Real-time Frequency Analysis

The `AudioVisualizer` component provides visual feedback during recording:

```tsx Recorder.tsx
export const AudioVisualizer: React.FC<AudioVisualizerProps> = ({
  audioStream,
  isRecording,
}) => {
  const canvasRef = useRef<HTMLCanvasElement>(null);
  const audioCtxRef = useRef<AudioContext | null>(null);
  
  useEffect(() => {
    if (!audioStream || !canvasRef.current) return;
    
    const canvas = canvasRef.current;
    const ctx = canvas.getContext("2d");
    
    const audioCtx = new (window.AudioContext || window.AudioContext)();
    const analyser = audioCtx.createAnalyser();
    const source = audioCtx.createMediaStreamSource(audioStream);
    
    source.connect(analyser);
    analyser.fftSize = 256;
    
    const bufferLength = analyser.frequencyBinCount;
    const dataArray = new Uint8Array(bufferLength);
    
    const drawBars = () => {
      animationFrameId = requestAnimationFrame(drawBars);
      analyser.getByteFrequencyData(dataArray);
      
      ctx.fillStyle = "#FFFFFF";
      ctx.fillRect(0, 0, canvas.width, canvas.height);
      
      const barWidth = (canvas.width / bufferLength) * 1.5;
      let x = 0;
      
      for (let i = 0; i < bufferLength; i++) {
        let barHeight = minBarHeight;
        if (isVisualizing) {
          barHeight = Math.max(
            minBarHeight,
            (dataArray[i] / 128.0) * (canvas.height / 2)
          );
        }
        
        ctx.fillStyle = `rgba(35, 43, 43)`;
        ctx.fillRect(
          x,
          canvas.height / 2 - barHeight / 2,
          barWidth - 1,
          barHeight
        );
        
        x += barWidth * 2.5;
      }
    };
    
    drawBars();
  }, [audioStream, isVisualizing]);
  
  return <canvas ref={canvasRef} className="w-800 h-100" />;
};
```
